Moist Guard

By : Muhammad Fa'iz Alaric

Electronics Year: 2026

Deskripsi Singkat: Project ini behasil dibuat dengan cara menyatukan beberapa komponen dengan sensor kelembapan. Dengan demikian para pengguna dapat menggunakan produk hasil project ini dengan cara kabel dimasukan ke aliran listrik. Sehingga tujuan dari produk ini yaitu agar Para pengguna tidak kesulitan saat menyiram tanaman karna sudah disiram dengan otomatis.

Project Features

  • Membantu Perawatan Tanaman
  • Solusi Hemat dan Fungsional
  • Inovasi Ramah Lingkungan
  • Pendidikan dan Kesadaran Energi
